Our Projects
We develop both Discord and web applications to help our users manage and enjoy their communities.
Kiai
Kiai is a next generation leveling bot designed for Discord servers who want to level up their community.
JT-99
JT-99 is a cross-server chat network with a dedicated team of moderators, designed to provide a safe and welcoming environment for all users.
Carbon
Carbon is a powerful and versatile framework designed for building HTTP-based Discord bots. It provides a robust set of tools and features that enable developers to create efficient and scalable bots with ease.
